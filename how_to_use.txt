1. Создать папку с проектом
2. Скачать и установить Node.js + yarn
3. Проверка в терминале установлен ли Node.js: 

             yarn -v

4. Инициализируем package.json: 

            yarn init

5. Необходимо загрузить Webpack v4 в качестве модуля и webpack-cli, чтобы запустить его из терминала:
           
           yarn add webpack webpack-cli -D

/* -D for development, т.е во время разработки включаются все зависимости*/
/* в рез-те выполнения данной команды появится папка node_modules*/

6. Создать папку src с файлом index.js

7. Далее откройте package.json и добавьте build-скрипт:

       		"scripts": {
	   	    "dev": "webpack --mode development",
		    "build": "webpack --mode production"
        	}

// Новый  Webpack не требует наличия конфигурации, поэтому перед началом использования вам не 
//нужно возиться с webpack.config.js. Из-за этого Webpack вынужден действовать по умолчанию, и 
//он будет искать папку ./src, а в ней index.js и выводить в ./dist/main.js; main.js — это ваш 
//скомпилированный файл с зависимостями.

8. Babel:
          yarn add @babel/core babel-loader babel-presets-env -D
